A browser extension to generate favicon if it doesn't exist.

![image](https://raw.githubusercontent.com/kaibadash/auto_favicon_gen_extention/master/doc/favicon.png)

## Setup

```
$ yarn
```

## Watch and build

```
yarn dev chrome
yarn dev firefox
yarn dev opera
yarn dev edge
```

## Build extension

```
yarn build chrome
yarn build firefox
yarn build opera
yarn build edge
```
